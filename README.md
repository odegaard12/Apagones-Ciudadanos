# Apagones Ciudadanos

Mapa ciudadano de incidencias eléctricas basado en reportes de usuarios.

## Principios
- Sin cuentas
- Sin CUPS
- Sin texto libre
- Sin fotos
- Sin mostrar viviendas exactas
- OpenStreetMap + reportes ciudadanos

## Stack
- Frontend: React + Vite + Leaflet
- Backend: FastAPI + SQLite
- Despliegue: Docker Compose

## Arranque
docker compose up --build -d

## URL
http://TU_IP:8098
